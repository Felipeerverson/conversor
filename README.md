# 💱 Conversor de Moedas em Java

Este projeto é um **Conversor de Moedas desenvolvido em Java**, que utiliza a API ExchangeRate para obter **cotações atualizadas em tempo real**.

O sistema permite converter valores entre diferentes moedas através de um **menu interativo no terminal**.

---

## 🚀 Tecnologias utilizadas

- Java
- IntelliJ IDEA
- ExchangeRate API
- Gson (biblioteca para leitura de JSON)

---

## 🌎 API utilizada

Este projeto utiliza a API pública:

https://www.exchangerate-api.com/

Ela fornece taxas de câmbio atualizadas entre diversas moedas do mundo.

---

## 📌 Funcionalidades

O sistema permite realizar as seguintes conversões:

1. Dólar → Real  
2. Real → Dólar  
3. Euro → Dólar  
4. Dólar → Euro  

O programa:

- consulta a API de câmbio
- recebe os dados em JSON
- interpreta os dados usando a biblioteca Gson
- realiza a conversão escolhida pelo usuário
